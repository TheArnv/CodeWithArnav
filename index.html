COMPLETE HTML

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of web pages using markup.

HTML History

Created by Tim Berners-Lee in 1991. HTML5 is the latest version, introduced in 2014 with semantic tags and multimedia support.

Why Learn HTML?

Foundation of web development. Every website uses HTML. Easy to learn, essential for all web developers.

What is CSS?

CSS (Cascading Style Sheets) is the language used to style HTML documents. It controls colors, layouts, fonts, and overall visual presentation.

CSS History

CSS was first proposed in 1994 by Håkon Wium Lie. CSS3 (current version) introduced modules, flexbox, grid, animations, and responsive design.

Why Learn CSS?

CSS makes websites beautiful and responsive. Essential for frontend development. Combines with HTML to create stunning user interfaces.

JavaScript Complete Reference

What is JavaScript?

JavaScript is a programming language that makes web pages interactive. It runs in the browser and can manipulate HTML, CSS, and handle user interactions.

History

Created by Brendan Eich in 1995. Originally called Mocha, then LiveScript, finally JavaScript. ECMAScript is the standardized version.

Why Learn JS?

Essential for frontend development. Works everywhere. Can be used for frontend, backend (Node.js), and mobile apps.

Variables

let, const, var

Variables
What are variables?

Variables are containers for storing data values. JavaScript has three ways to declare variables.

// let - can be reassigned (block scope)
let age = 25;
age = 26; // ✅ Allowed

// const - cannot be reassigned (block scope)
const PI = 3.14;
// PI = 3.15; // ❌ Error

// var - old way (function scope)
var old = "deprecated";

Data Types

Types
What are data types?

Data types specify what kind of value a variable holds. JavaScript has primitive and reference types.

// Primitive Types
let str = "Hello"; // String
let num = 42; // Number
let bool = true; // Boolean
let und; // Undefined
let n = null; // Null

// Reference Types
let obj = {}; // Object
let arr = []; // Array
let fn = function() {}; // Function

Type Conversion

Conversion
What is type conversion?

Converting values from one data type to another.

// Explicit conversion
String(123); // "123"
Number("123"); // 123
Boolean(0); // false

// parseInt/parseFloat
parseInt("123px"); // 123
parseFloat("3.14"); // 3.14

Operators

Arithmetic Operators

Math
What are arithmetic operators?

Perform mathematical calculations on numbers.

let a = 10, b = 3;

a + b; // 13 (Addition)
a - b; // 7 (Subtraction)
a * b; // 30 (Multiplication)
a / b; // 3.33 (Division)
a % b; // 1 (Modulus - remainder)
a ** b; // 1000 (Exponentiation)

a++; // Increment
b--; // Decrement

Assignment Operators

Assign
What are assignment operators?

Assign values to variables, often with a calculation.

let x = 10; // = assignment
x += 5; // x = x + 5 (15)
x -= 3; // x = x - 3 (12)
x *= 2; // x = x * 2 (24)
x /= 4; // x = x / 4 (6)
x %= 4; // x = x % 4 (2)
x **= 3; // x = x ** 3 (8)

Comparison Operators

Compare
What are comparison operators?

Compare two values and return a boolean (true/false).

let a = 5, b = "5";

a == b; // true (loose equality - value only)
a === b; // false (strict equality - value AND type)
a != b; // false (loose inequality)
a !== b; // true (strict inequality)

a > 3; // true
a < 10; // true
a >= 5; // true
a <= 4; // false

Logical Operators

Logic
What are logical operators?

Combine or invert boolean values for complex conditions.

let a = true, b = false;

a && b; // false (AND - both must be true)
a || b; // true (OR - at least one true)
!a; // false (NOT - inverts)

// Short-circuit evaluation
let name = "" || "Guest"; // "Guest"
let user = "John" && "Admin"; // "Admin"

Ternary Operator

Conditional
What is the ternary operator?

A shorthand for if-else statements. Syntax: condition ? valueIfTrue : valueIfFalse

let age = 18;
let status = age >= 18 ? "Adult" : "Minor";
// status = "Adult"

typeof / instanceof

Type Checking
What are typeof and instanceof?

Operators to check the type of a value or object.

typeof "hello"; // "string"
typeof 42; // "number"
typeof true; // "boolean"
typeof undefined; // "undefined"
typeof null; // "object" (bug)
typeof {}; // "object"

[] instanceof Array; // true

Control Flow

if-else

Conditionals
What is if-else?

Executes code blocks based on boolean conditions.

let age = 18;

if (age >= 18) {
  console.log("Adult");
} else if (age >= 13) {
  console.log("Teen");
} else {
  console.log("Child");
}

switch

Conditionals
What is switch?

Evaluates an expression and matches it against multiple cases.

let day = "Monday";

switch(day) {
  case "Monday":
    console.log("Work day");
    break;
  case "Saturday":
  case "Sunday":
    console.log("Weekend");
    break;
  default:
    console.log("Other day");
}

for loop

Loops
What is a for loop?

Iterates a block of code a specified number of times.

for (let i = 0; i < 5; i++) {
  console.log(i);
}
// 0, 1, 2, 3, 4

while loop

Loops
What is a while loop?

Executes a block while a condition is true.

let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

// do-while (executes at least once)
let j = 0;
do {
  console.log(j);
  j++;
} while (j < 5);

break / continue

Loop Control
What are break and continue?

Control loop execution - break exits the loop, continue skips to next iteration.

for (let i = 0; i < 10; i++) {
  if (i === 3) continue; // skip 3
  if (i === 7) break; // stop at 7
  console.log(i);
}
// 0,1,2,4,5,6

Functions

Function Declaration

Functions
What is a function declaration?

A named function that is hoisted (can be called before declaration).

function greet(name) {
  return "Hello " + name;
}

greet("John"); // "Hello John"

Function Expression

Functions
What is a function expression?

A function stored in a variable. Not hoisted - must be defined before use.

const greet = function(name) {
  return "Hello " + name;
};

greet("John"); // "Hello John"

Arrow Function

Functions
What is an arrow function?

ES6 concise syntax for functions. Doesn't have its own 'this'.

const greet = (name) => {
  return "Hello " + name;
};

// Single line implicit return
const square = x => x * x;
const hello = () => "Hello World";

Parameters & Arguments

Functions
What are parameters and arguments?

Parameters are placeholders in function definition. Arguments are actual values passed.

// Default parameters
function greet(name = "Guest") {
  return "Hello " + name;
}

// Rest parameters
function sum(...numbers) {
  return numbers.reduce((a,b) => a + b, 0);
}
sum(1,2,3,4); // 10

Callback Functions

Functions
What is a callback?

A function passed as an argument to another function, to be executed later.

function process(data, callback) {
  let result = data.toUpperCase();
  callback(result);
}

process("hello", output => console.log(output));
// "HELLO"

Closures

Advanced
What is a closure?

A function that remembers its lexical scope even when called outside that scope.

function counter() {
  let count = 0; // private variable
  return function() {
    return ++count;
  };
}

let c = counter();
c(); // 1
c(); // 2
c(); // 3

Arrays

push / pop

Arrays
What do push() and pop() do?

push() adds to end. pop() removes from end. Both modify original array.

let fruits = ["Apple", "Banana"];

fruits.push("Orange"); // ["Apple","Banana","Orange"]
let last = fruits.pop(); // "Orange", fruits=["Apple","Banana"]

shift / unshift

Arrays
What do shift() and unshift() do?

unshift() adds to beginning. shift() removes from beginning.

let fruits = ["Apple", "Banana"];

fruits.unshift("Orange"); // ["Orange","Apple","Banana"]
let first = fruits.shift(); // "Orange", fruits=["Apple","Banana"]

splice / slice

Arrays
What are splice() and slice()?

splice() changes array content. slice() returns a copy.

let arr = [1,2,3,4,5];

// splice(start, deleteCount, ...items)
arr.splice(2, 1, 'a'); // [1,2,'a',4,5]

// slice(start, end)
let sliced = arr.slice(1, 3); // [2,'a']

map()

Arrays
What does map() do?

Creates new array by transforming every element. Does not modify original.

let numbers = [1, 2, 3, 4];

let doubled = numbers.map(x => x * 2);
// doubled = [2, 4, 6, 8]

filter()

Arrays
What does filter() do?

Creates new array with elements that pass a condition.

let numbers = [1, 2, 3, 4, 5, 6];

let evens = numbers.filter(x => x % 2 === 0);
// evens = [2, 4, 6]

reduce()

Arrays
What does reduce() do?

Reduces array to a single value by applying a function cumulatively.

let numbers = [1, 2, 3, 4, 5];

let sum = numbers.reduce((acc, curr) => acc + curr, 0);
// sum = 15

find() / findIndex()

Arrays
What do find() and findIndex() do?

find() returns first matching element. findIndex() returns its index.

let numbers = [1, 2, 3, 4, 5];

let found = numbers.find(x => x > 3); // 4
let index = numbers.findIndex(x => x > 3); // 3

includes() / indexOf()

Arrays
What do includes() and indexOf() do?

includes() checks existence. indexOf() returns first index or -1.

let fruits = ["Apple", "Banana", "Orange"];

fruits.includes("Banana"); // true
fruits.indexOf("Banana"); // 1
fruits.indexOf("Grape"); // -1

sort() / reverse()

Arrays
What do sort() and reverse() do?

sort() sorts array. reverse() reverses order.

let numbers = [3, 1, 4, 2, 5];

numbers.sort((a,b) => a - b); // [1,2,3,4,5]
numbers.reverse(); // [5,4,3,2,1]

concat() / join()

Arrays
What do concat() and join() do?

concat() merges arrays. join() creates string from array.

let arr1 = [1,2];
let arr2 = [3,4];

let combined = arr1.concat(arr2); // [1,2,3,4]
let joined = arr1.join("-"); // "1-2"

Objects

Object Methods

Objects
What are object methods?

Built-in methods to work with objects.

Object.keys(obj); // ["a", "b", "c"]
Object.values(obj); // [1, 2, 3]
Object.entries(obj); // [["a",1], ["b",2], ["c",3]]

Spread & Destructure

ES6
What are spread and destructuring?

ES6 features to easily copy objects and extract properties.

// Spread
let obj1 = {a:1, b:2};
let obj2 = {...obj1, c:3}; // {a:1,b:2,c:3}

// Destructuring
let {a, b} = obj1;
console.log(a); // 1

Syntactic sugar over prototypes for object-oriented programming.

let p = new Person("John");

Numbers & Math

Number methods

Event Object

Events
Event object

Information about the event passed to handler.

element.addEventListener('click', (e) => {
  e.target; // clicked element
  e.currentTarget; // listener element
  e.preventDefault(); // prevent default
  e.stopPropagation(); // stop bubbling
  e.clientX, e.clientY; // mouse position
  e.key; // key pressed (keyboard)
});
